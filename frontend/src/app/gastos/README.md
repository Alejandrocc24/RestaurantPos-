# Componente de Gastos

Este componente permite la gestión de gastos y proveedores de la heladería.

## Funcionalidades

- **Gestión de Gastos**: Registro y administración de todos los gastos de la heladería
- **Gestión de Proveedores**: Administración de la información de proveedores

## Estructura

- `gastos.component.ts` - Lógica del componente
- `gastos.component.html` - Template HTML
- `gastos.component.css` - Estilos CSS
- `index.ts` - Archivo de exportación

## Uso

El componente se accede desde el sidebar principal a través del botón "Gastos" y muestra dos opciones principales:

1. **Gastos**: Para gestionar los gastos de la heladería
2. **Proveedores**: Para administrar la información de proveedores

## Navegación

El componente está integrado en el sistema de navegación del dashboard y se puede acceder a través de la ruta `/gastos`.
